HobbyHorse Кавказ Арена — демонстрационный прототип

Как запустить:
1. Распакуйте архив.
2. Откройте файл index.html в браузере.

Что есть в прототипе:
- выбор региона игрока;
- остальные регионы управляются ботами;
- пошаговые раунды;
- события и кризисы;
- ресурсы регионов;
- общий этап округа;
- подготовка Кубка Кавказа по хоббихорсингу;
- несколько вариантов победы и общий проигрыш.

Файлы:
- index.html — структура страницы;
- style.css — оформление;
- script.js — логика игры.


Обновление логики:
- Кубок теперь проводится как отдельный финальный этап после 8-го раунда.
- Для проведения Кубка нужна готовность минимум 60.
- Если готовность 80+, пройти этапы проще, но 100 не обязательно.
- Кубок состоит из четырёх этапов: гонки, препятствия, стиль, командные заезды.
- Общий ресурс ограничен лимитом 12 и автоматически тратится на поддержку кризисов или подготовку Кубка.


Обновление v3:
- У регионов разные стартовые ресурсы и разные профили ресурсов.
- Доход теперь зависит от экономики, туризма, активности, стабильности и кризисов.
- Боты ходят не разом, а по очереди.
- В интерфейсе показывается, какой бот сейчас ходит.
- В журнале видно: какой регион-бот получил событие и какое действие выбрал.


Обновление v4:
- Кубок теперь нельзя провести при неполной шкале готовности.
- Для проведения Кубка нужно ровно 100% готовности.
- Если после 8-го раунда готовность ниже 100%, игра показывает общий проигрыш по ключевой цели.


Обновление v5:
- Исправлена подпись на кнопке подготовки Кубка.
- В интерфейсе теперь указано: для проведения Кубка нужно 100% готовности.


Обновление v6:
- Исправлена финальная проверка: при готовности Кубка ниже 100% победа больше невозможна.
- Если готовность 84%, 90% или 99%, игра показывает общий проигрыш по ключевой цели.
- Финальные этапы Кубка запускаются только после достижения 100%.
