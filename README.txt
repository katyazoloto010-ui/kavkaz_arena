HobbyHorse Кавказ Арена
=======================

Это полноценный интерактивный веб-прототип кооперативной стратегической игры.

Как запустить:
1. Распакуйте архив.
2. Откройте файл index.html в браузере.
3. Никакой сервер, база данных или установка не нужны.

Что реализовано:
- главный экран проекта;
- выбор одного из 7 регионов СКФО;
- разные стартовые показатели регионов;
- система ходов на 12 этапов;
- карточки событий, кризисов, общественных инициатив и межрегиональных проектов;
- изменение показателей в реальном времени;
- действия игрока: культурная инициатива, межрегиональный проект;
- главное событие: Кубок Кавказа по хоббихорсингу;
- условия победы и поражения;
- финальный экран с итогами;
- журнал игры;
- адаптивный интерфейс для компьютера и мобильного экрана.

Файлы:
- index.html — структура сайта;
- style.css — оформление интерфейса;
- script.js — игровая логика;
- README.txt — описание проекта.

Для диплома можно описывать проект как:
«Интерактивный веб-прототип кооперативной стратегической игры “HobbyHorse Кавказ Арена”, демонстрирующий базовые игровые механики, систему карточек, развитие регионов, кризисы и финальное культурно-спортивное событие».
